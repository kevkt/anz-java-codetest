# ANZ Java Code Test

## Install
### Dependencies
* Java 11
* Gradle

## Build and run locally
```
gradle clean build

```

Run a test server
```
gradle bootRun
``` 

## Run tests
The code includes some intial tests of the API endpoints only.

```
gradle test
``` 

### Postman collection
A Postman collection has been provided to call the two services implemented.
